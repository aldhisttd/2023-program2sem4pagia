Pilihan Soal

  1. Seri soal ada 2, GENAP dan GANJIL
  2. Silahkan pilih soal sesuai dengan angka digit terakhir NIM kalian, Ganjil atau Genap
  3. Untuk angka digit akhir 0, pilih yang GENAP

Instruksi Soal

  1. Buat dulu branch anda dari branch master
  2. Ketentuan nama branch adalah: perbaikan/ganjil_NIM atau quiz3/genap_NIM
  3. Lokasi project kalian beradan di folder quiz/perbaikan/01_crud/NIM
  4. Buat database sesuai dengan rancangan tabel yang sudah di tentukan
  5. Lalu buat modul READ, UPDATE, DELETE sesuai dengan materi di folder materi/crud
  6. Lakukan PR dengan nama branch tadi dan jangan lupa untuk melampirkan hasil output dari program
  7. Jangan lupa di PR nya untuk menambahkan saya sebagai reviewer, ada di bagian kanan ketika PR
  8. Ikuti semua instruksi dengan benar. Tertib dan kerjakan sendiri, semua akhtivitas kalian terpantau CCTV.

Ganjil

Tabel = mahasiswa
Nama Field Tipe Data Jenis Input
nim string input text
nama string input text
jurusan string select option (novel, komik, kamus)
umur number number
photo string input file (upload file photo)

Genap

Tabel = kendaraan
Nama Field Tipe Data Jenis Input
kode string input text
tanggal_masuk date input date
jenis_kendaraan string select option (SUV, City Car, Sedan)
harga number number
photo string input file (upload file gambar)
