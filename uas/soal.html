Pilihan Soal

  1. Seri soal ada 2, GENAP dan GANJIL
  2. Silahkan pilih soal sesuai dengan angka digit terakhir NIM kalian, Ganjil atau Genap
  3. Untuk angka digit akhir 0, pilih yang GENAP

Instruksi

  1. Source branch = master
  2. Ketentuan nama branch : uas/NIM
  3. Jika sudah selesai lakukan PR dengan judul PR : uas/crud/NIM/NAMA_LENGKAP
  4. Mention saya di bagian description PR kalian
  5. Add saya sebagai Reviewer di PR kalian

Soal

  1. Buatlah full aplikasi sederhana CRUD + LoginDB dari data di bawah
  2. Lokasi project berada di folder uas/NIM
  3. Nama database harus sesuai dengan NIM kalian masing-masing
  4. Table Name, Field Name & Data Type harus sesuai dan identik dengan data table kalian
  5. Input Type mengacu pada jenis form
  6. Jangan lupa screenshot aplikasi dan database nya dan lampirkan di bagian description
